## First Launch and Login

When you first launch Bodeguita, you'll be greeted with an elegant login screen featuring a gradient background and animated shopping cart logo.

<Steps>
  <Step title="Open Bodeguita">
    Launch the application from your device's app drawer.
  </Step>
  
  <Step title="Enter Credentials">
    Use the default admin credentials to log in:
    
    - **Username**: `admin`
    - **Password**: `1234`
    
    ```kotlin app/src/main/java/com/trabajo/minitienda/screens/LoginScreen.kt
    // Default login credentials
    private val DEFAULT_PASSWORD = "1234"
    
    // Login validation
    if (username == "admin" && password == savedPassword) {
        navController.navigate("dashboard") {
            popUpTo("login") { inclusive = true }
        }
    }
    ```
  </Step>
  
  <Step title="First Login">
    Click the **Ingresar** button to access the dashboard. The app will validate your credentials and navigate you to the main panel.
  </Step>
</Steps>

<Warning>
For security purposes, it's highly recommended to change the default password immediately after your first login. Click "Cambiar contraseña de admin" on the login screen to set a new password.
</Warning>

### Weekly Activity Chart

Below the metrics, you'll find a visual chart showing your weekly sales activity, helping you identify trends and peak sales periods.

### Low Stock Alerts

The dashboard automatically monitors your inventory and displays a warning banner for products with stock below 10 units:

```kotlin app/src/main/java/com/trabajo/minitienda/screens/DashboardScreen.kt
val lowStockItems = products
    .filter { it.stock < 10 }
    .sortedBy { it.stock }
    .map { it.name to it.stock }

if (lowStockItems.isNotEmpty()) {
    DashboardLowStockBanner(
        lowStockItems = lowStockItems,
        onSeeProducts = { navController.navigate("products") }
    )
}
```

<Note>
Click "Ver todos" on the low stock banner to quickly navigate to the Products module and restock items.
</Note>

### Quick Actions

At the bottom of the dashboard, you'll find four quick action cards:

- **Productos**: Manage your inventory
- **Ventas**: Record new sales
- **Compras**: Register purchases
- **Cierre de Caja**: View cash closure reports

## Quick Tour of Main Features

### 1. Managing Products

<Steps>
  <Step title="Navigate to Products">
    Open the navigation drawer and tap **Productos**, or click the "Productos" quick action card on the dashboard.
  </Step>
  
  <Step title="View Product List">
    You'll see a list of all registered products with their stock levels, prices, and categories. Use the search bar to find specific products by name or category.
  </Step>
  
  <Step title="Add a New Product">
    Tap the **+** floating action button to open the product registration screen. Fill in:
    - Product name
    - Category
    - Initial stock quantity
    - Purchase price
    - Sale price
  </Step>
  
  <Step title="Save Product">
    Click **Guardar** to add the product to your inventory. The product will immediately appear in your product list and affect dashboard metrics.
  </Step>
</Steps>

<Info>
**Product Features**:
- Assign categories for better organization
- Set different purchase and sale prices
- Track stock levels automatically
- Edit or delete products as needed
- Search products by name or category
</Info>

### 2. Recording Sales

<Steps>
  <Step title="Navigate to Sales">
    From the navigation drawer, select **Ventas** or use the quick action card.
  </Step>
  
  <Step title="Select Products">
    Browse or search for products you want to sell. Tap on a product to add it to the current sale.
  </Step>
  
  <Step title="Set Quantities">
    For each product, specify the quantity being sold. The system will automatically:
    - Calculate line item totals
    - Update the running subtotal
    - Check available stock
    - Calculate taxes and final total
  </Step>
  
  <Step title="Complete Sale">
    Review the sale details and tap **Confirmar Venta** to finalize. The system will:
    - Deduct sold items from inventory
    - Record the transaction with date and time
    - Update dashboard metrics
    - Make the sale available in reports
  </Step>
</Steps>

<Warning>
The system prevents overselling. If you try to sell more items than available in stock, you'll receive an error message and the sale will be blocked.
</Warning>

### 3. Registering Purchases

<Steps>
  <Step title="Navigate to Purchases">
    Open the navigation drawer and select **Compras**.
  </Step>
  
  <Step title="Select Supplier">
    Choose the supplier you're purchasing from. If the supplier doesn't exist, you can add them from the Suppliers module.
  </Step>
  
  <Step title="Add Products to Purchase">
    Select products and specify quantities being purchased. The system calculates the total cost based on purchase prices.
  </Step>
  
  <Step title="Complete Purchase">
    Confirm the purchase to:
    - Add purchased quantities to inventory
    - Record the transaction
    - Link the purchase to the supplier
    - Update cash closure reports
  </Step>
</Steps>

### 4. Viewing Cash Closure

<Steps>
  <Step title="Navigate to Cash Closure">
    From the drawer menu, tap **Cierre** or use the dashboard quick action.
  </Step>
  
  <Step title="Review Daily Summary">
    The cash closure screen displays:
    - Total sales for the day
    - Total purchases for the day
    - Gross income from sales
    - Expenses from purchases
    - Net profit/loss calculation
  </Step>
  
  <Step title="Filter by Date">
    Use the date picker to view cash closure reports for specific days, helping you track performance over time.
  </Step>
</Steps>

<Info>
Cash closure reports are generated automatically based on your sales and purchase transactions. No manual entry required!
</Info>

## Navigation Structure

Understand how the app's navigation flows:

```kotlin app/src/main/java/com/trabajo/minitienda/screens/MainNavigation.kt
NavHost(navController = navController, startDestination = "login") {
    composable("login") { LoginScreen(navController) }
    
    composable("dashboard") {
        DashboardScreen(
            navController = navController,
            productViewModel = productViewModel,
            salesViewModel = salesViewModel,
            dashboardViewModel = dashboardViewModel,
            onMenuClick = openDrawer
        )
    }
    
    composable("products") {
        ProductListScreen(navController, productViewModel, onMenuClick = openDrawer)
    }
    
    composable("product_registration") {
        ProductRegistrationScreen(
            navController, productViewModel, categoryViewModel,
            onMenuClick = openDrawer
        )
    }
    
    composable("sales") {
        SalesScreen(navController, salesViewModel, onMenuClick = openDrawer)
    }
    
    composable("purchases") {
        PurchasesScreen(
            navController = navController,
            purchasesVM = purchasesViewModel,
            onMenuClick = openDrawer
        )
    }
    
    composable("cash_closure") {
        CashClosureScreen(navController, vm = cashClosureVM, onMenuClick = openDrawer)
    }
    
    composable("profile") {
        ProfileScreen(
            navController, onMenuClick = openDrawer, themeManager = themeManager
        )
    }
}
```

## Common Workflows

### Daily Opening Routine

1. Log in to Bodeguita
2. Check the dashboard for low stock alerts
3. Review yesterday's cash closure
4. Restock items as needed via Purchases
5. Start recording sales

### Adding New Products

1. Navigate to Products module
2. Click the + button
3. Fill in product details
4. Assign a category
5. Set initial stock and prices
6. Save the product

### Processing a Sale

1. Go to Sales module
2. Search and add products to cart
3. Set quantities for each item
4. Review total amount
5. Confirm the sale

### Recording Supplier Purchases

1. Navigate to Purchases
2. Select or add a supplier
3. Choose products and quantities
4. Verify total cost
5. Complete the purchase
