## What is Bodeguita?

Bodeguita is a mobile application developed in **Android Studio with Kotlin and Jetpack Compose** that enables **management of sales, purchases, products, suppliers, and cash closure** for local stores. The application uses **Room** as its local database and follows the **MVVM architecture** for clean, scalable, and maintainable code.

<Info>
Bodeguita is designed specifically for small to medium-sized retail stores looking to digitize their inventory and sales management processes.
</Info>

## Technology Stack

Bodeguita is built with modern Android development technologies:

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose with Material Design 3
- **Database**: Room (SQLite)
- **Architecture**: MVVM (Model-View-ViewModel)
- **Navigation**: Jetpack Navigation Compose
- **Charts**: Vico library for data visualization
- **Minimum SDK**: Android 7.1 (API 25)
- **Target SDK**: Android 14 (API 36)

```kotlin app/build.gradle.kts
    }
}
```

## Core Architecture

The application follows a clean MVVM architecture pattern:

```kotlin app/src/main/java/com/trabajo/minitienda/data/database/AppDatabase.kt
@Database(
    entities = [
        Product::class,
        Category::class,
        Sale::class,
        SaleDetail::class,
        Supplier::class,
        Purchase::class,
        PurchaseDetail::class,
        MovimientoInventario::class
    ],
    version = 8,
    exportSchema = true
)
abstract class AppDatabase : RoomDatabase() {
    abstract fun productDao(): ProductDao
    abstract fun categoryDao(): CategoryDao
    abstract fun saleDao(): SaleDao
    abstract fun supplierDao(): SupplierDao
    abstract fun purchaseDao(): PurchaseDao
    abstract fun movementDao(): MovementDao
}
```

<Note>
The database uses Room's migration strategy with `fallbackToDestructiveMigration()` for development flexibility.
</Note>

## Main Modules

### Products Module
- Register, edit, and delete products
- Assign categories to each product
- Display available stock and current price
- Product search by name or category

### Purchases Module
- Register purchases from suppliers
- Add multiple products in a single purchase
- Automatically calculate total cost
- Update stock for purchased products

### Sales Module
- Register sales with multiple products
- Calculate subtotal, taxes, and total amount
- Control inventory when making a sale
- View sales history with date and time

### Cash Closure
- Display daily summary of sales and purchases
- Calculate income, expenses, and net profit
- Record the final amount of the day (cash register)

### Suppliers Module
- Register and maintain supplier information
- Link suppliers with purchases made

### Dashboard
- Visualize summarized store statistics
- Show totals for sales, purchases, and active products
- Quick access to main modules
