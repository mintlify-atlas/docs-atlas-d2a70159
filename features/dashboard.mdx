The Dashboard is the central hub of Bodeguita, providing real-time metrics, weekly activity charts, low stock alerts, and quick navigation to core modules.

## Overview

The Dashboard screen (`DashboardScreen.kt`) displays key business metrics and provides instant access to all major features. It combines data from multiple ViewModels to present a comprehensive view of store operations.

### Key Components

- **Real-time Metrics**: Today's sales, units sold, total products, and last sale amount
- **Weekly Activity Chart**: Visual representation of sales over the last 7 days
- **Low Stock Alerts**: Automatic detection of products below threshold (< 10 units)
- **Quick Actions**: One-tap navigation to Products, Sales, Purchases, and Cash Closure

## Architecture

The Dashboard follows MVVM pattern with reactive data flows:

## Real-Time Metrics

The dashboard displays four primary metrics that update automatically:

### Today's Sales Count

Tracks the number of transactions completed today:

```kotlin
// DashboardViewModel.kt
val todaySalesCount: StateFlow<Int> =
    saleDao.todaySalesCount()
        .stateIn(viewModelScope, SharingStarted.WhileSubscribed(5_000), 0)
```

### Units Sold Today

Calculates total quantity of all items sold:

```kotlin
// DashboardViewModel.kt
val todayUnitsSold: StateFlow<Int> =
    saleDao.todayUnitsSold()
        .stateIn(viewModelScope, SharingStarted.WhileSubscribed(5_000), 0)
```

### Total Products

Counts all registered products in inventory:

```kotlin
// DashboardScreen.kt:58
val products = productViewModel.products.collectAsState(initial = emptyList()).value
DashboardStatsGrid(
    todaySales = todaySales,
    todayUnits = todayUnits,
    lastSale   = lastSale,

```

### Last Sale Information

Displays the most recent sale with ID, total, and timestamp:

```kotlin
// DashboardViewModel.kt
val lastSaleBrief: StateFlow<SaleBrief?> =
    saleDao.lastSaleBrief()
        .stateIn(viewModelScope, SharingStarted.WhileSubscribed(5_000), null)
```

## Weekly Activity Chart

The dashboard displays sales trends over the last 7 days using the Vico charting library:

```kotlin
// DashboardScreen.kt:67
ActividadSemanalChart(salesViewModel = salesViewModel)
```

### Chart Data Flow

<Steps>
  <Step title="ViewModel Loads Weekly Data">
    ```kotlin
    // SalesViewModel.kt:92
    private fun loadWeeklySales() {
        val calendar = Calendar.getInstance().apply {
            add(Calendar.DAY_OF_YEAR, -6)
            set(Calendar.HOUR_OF_DAY, 0)
            set(Calendar.MINUTE, 0)
            set(Calendar.SECOND, 0)
            set(Calendar.MILLISECOND, 0)
        }
        val sevenDaysAgoTimestamp = calendar.timeInMillis

        viewModelScope.launch {
            repo.getWeeklySalesSummary(sevenDaysAgoTimestamp)
                .catch { exception ->
                    _weeklySales.value = emptyList()
                }
                .collect { salesData ->
                    _weeklySales.value = salesData
                }
        }
    }
    ```
  </Step>

  <Step title="Chart Processes Data">
    ```kotlin
    // ActividadSemanalChart.kt:51
    val chartData = remember(weeklySalesData) {
        val salesMap = weeklySalesData.associate { it.saleDate to it.total }
        val entries = mutableListOf<ChartEntry>()
        val days = mutableListOf<String>()

        val dbFormat = SimpleDateFormat("yyyy-MM-dd", Locale.getDefault())
        val labelFormat = SimpleDateFormat("E", Locale("es", "ES"))

        val calendar = Calendar.getInstance().apply {
            add(Calendar.DAY_OF_YEAR, -6)
        }

        repeat(7) { i ->
            val dateKey = dbFormat.format(calendar.time)
            val dayLabel = labelFormat.format(calendar.time)
                .replaceFirstChar { it.titlecase(Locale.getDefault()) }

            val total = salesMap[dateKey] ?: 0.0
            entries.add(entryOf(i, total.toFloat()))
            days.add(dayLabel)

            calendar.add(Calendar.DAY_OF_YEAR, 1)
        }

        Pair(entries, days)
    }
    ```
  </Step>

  <Step title="Render Chart">

## Low Stock Alerts

The dashboard automatically detects products with inventory below 10 units and displays a warning banner:

```kotlin
// DashboardScreen.kt:70
val lowStockItems = products
    .filter { it.stock < 10 }
    .sortedBy { it.stock }
    .map { it.name to it.stock }

if (lowStockItems.isNotEmpty()) {
    DashboardLowStockBanner(
        lowStockItems = lowStockItems,
        onSeeProducts = { navController.navigate("products") }
    )
}
```

### Low Stock Banner UI

```kotlin
// DashboardScreen.kt:221
@Composable
private fun DashboardLowStockBanner(
    lowStockItems: List<Pair<String, Int>>,
    onSeeProducts: () -> Unit,
    maxItems: Int = 3
) {
    val bg = WarningColor.copy(alpha = 0.06f)
    val br = WarningColor.copy(alpha = 0.25f)

    AppCard(
        modifier = Modifier
            .fillMaxWidth()
            .border(1.dp, br, RoundedCornerShape(16.dp))
            .background(bg, RoundedCornerShape(16.dp))
            .padding(horizontal = 12.dp, vertical = 10.dp)
    ) {
        Column(verticalArrangement = Arrangement.spacedBy(8.dp)) {
            Row(
                modifier = Modifier.fillMaxWidth(),
                verticalAlignment = Alignment.CenterVertically
            ) {
                Row(
                    verticalAlignment = Alignment.CenterVertically,
                    modifier = Modifier.weight(1f)
                ) {
                    Icon(
                        imageVector = Icons.Default.ErrorOutline,
                        contentDescription = null,
                        tint = WarningColor,
                        modifier = Modifier.size(18.dp)
                    )
                    Spacer(Modifier.width(8.dp))
                    Text(
                        "Productos con Bajo Stock",
                        style = MaterialTheme.typography.titleSmall,
                        maxLines = 1,
                        overflow = TextOverflow.Ellipsis
                    )
                }
                TextButton(onClick = onSeeProducts) {
                    Text("Ver todos", color = WarningColor)
                }
            }

            Column(verticalArrangement = Arrangement.spacedBy(6.dp)) {
                lowStockItems.take(maxItems).forEach { (name, qty) ->
                    Row(
                        modifier = Modifier.fillMaxWidth(),
                        horizontalArrangement = Arrangement.SpaceBetween
                    ) {
                        Text(name, style = MaterialTheme.typography.bodySmall)
                        Text(
                            text = "$qty unidades",
                            style = MaterialTheme.typography.bodySmall,
                            color = WarningColor
                        )
                    }
                }
            }
        }
    }
}
```

<Note>
  The low stock threshold is hardcoded at 10 units. Products with exactly 0 stock show as "Sin Stock" (red badge), while 1-9 units show as "Stock Bajo" (yellow badge).
</Note>

## Quick Action Cards

Provide one-tap navigation to major features:

```kotlin
// DashboardScreen.kt:305
@Composable
private fun DashboardQuickActionsGrid(onClick: (String) -> Unit) {
    val actions = listOf(
        DashboardAction(
            badge = "Gestionar",

            subtitle = "Ir a productos",
            route = "products",
            icon = Icons.Default.Inventory

        DashboardAction(

            title = "Cierre de Caja",
            subtitle = "Ir a cierre",

        )

        Text(text = "Acceso Rápido", style = MaterialTheme.typography.titleLarge)

            actions.chunked(1).forEach { row ->
                Row(Modifier.fillMaxWidth()) {

                    }
                }

        }
    }
```

## Best Practices

<Warning>
  **Performance Consideration**: All StateFlows use `SharingStarted.WhileSubscribed(5_000)` to automatically stop upstream flows when no collectors are active for 5 seconds, optimizing battery and memory usage.
</Warning>

<Info>
  The dashboard automatically refreshes when:
  - A new sale is completed
  - Product stock changes
  - Products are added or removed
  - Navigation returns to the dashboard
</Info>
